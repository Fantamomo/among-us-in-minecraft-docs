Want to try the plugin before setting it up yourself? A public demo server is available at `mc.fantamomo.com`.

<Steps>
    1. Open Minecraft and select the latest version (`1.21.11`).
    2. Go to **Multiplayer** and click **Add Server**.
    3. Enter `mc.fantamomo.com` as the server address and connect.
    4. Run `/au create test` to create a new game.
    5. Share the game code shown in the scoreboard with your friends — they can join with `/au join <code>`.
    6. Use `/au settings` to customize the game to your liking.
    7. Start the game with `/au start` and have fun!
</Steps>

:::tip[Not enough players?]
Run `/au settings`, click the redstone item, and set **Win Condition Check** to `false`. This lets you start the game with fewer players than normally required, but the game will no longer end automatically.

To re-enable automatic win detection, go back into `/au settings` and set **Win Condition Check** to `true`.
:::

:::note[Map]
The map used on this server is the [Polus – Among Us](https://www.planetminecraft.com/project/polus-among-us-4875493/) build from Planet Minecraft. It has been modified to work with this plugin and fit the server's setup.
:::
