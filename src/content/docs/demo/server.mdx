Want to try the plugin before setting it up yourself? A public demo server is available at `mc.fantamomo.com`.

<Steps>
    1. Open Minecraft and select the version `1.21.11`.
    2. Go to **Multiplayer** and click **Add Server**.
    3. Enter `mc.fantamomo.com` as the server address and connect.
    4. Run `/au create test` to create a new game.
    5. Invite your friends with `/au invite player>` — they will receive a message to join your game.
        Alternatively, share the game code displayed in the scoreboard — your friends can join with `/au join code>`.
        :::note[Chat restrictions]
        See [here](/gameplay/chat) for more information about chat restrictions.
        :::
    6. Use `/au settings` to customize the game to your liking.
    7. Start the game with `/au start` and have fun!
</Steps>

:::caution[Privacy]
**What data is collected when you join the demo server?**

When you connect to the server, the following data is processed:

- Minecraft username
- Minecraft UUID
- IP address (as with any standard Minecraft server)

The IP address is stored in the server logs. This information is **not shared with any third parties** and is used solely for operating and maintaining the server.

**Chat data**

If you send messages in chat, they will be stored in the server logs.

Additionally:

- While in a game, chat messages may also be recorded by the Action Log System: [Action Log System](/misc/action-log)
- If bots are present in the game, chat messages may be included in prompts sent to an AI system (Hackclub AI, via OpenRouter). These providers may process and potentially store the messages.

All data is used strictly for gameplay functionality, logging, and bot interaction purposes, and is not used for any unrelated processing.
:::

---

:::danger[Cannot connect]
If you are unable to connect to the demo server, there are two possible reasons:

1. **The server is restarting** — Please wait a few minutes and try again.
2. **DNS resolution failure** — In some cases, the domain may not resolve correctly. If this happens, try connecting directly via `mc.fantamomo.com:30685`.
:::

:::tip[Not enough players?]
**Play with bots**

Use `/au bot add <name>` to add a bot with the given name.
Read more about bots here: [Bots](/bots/introduction).

**Without bots**

Run `/au settings`, click the redstone item, and set **Win Condition Check** to `false`.
This lets you start the game with fewer players than normally required, but the game will no longer end automatically.

Note that if you start the game along, you will be always get a imposter role.

To re-enable automatic win detection, go back into `/au settings` and set **Win Condition Check** to `true`.
:::

:::note[Map]
The map used on this server is the [Polus – Among Us](https://www.planetminecraft.com/project/polus-among-us-4875493/) build from Planet Minecraft. It has been modified to work with this plugin and fit the server's setup.
:::

:::note[Version]
The demo server is running the latest version of the plugin, which is always the version you would get when building from source.
However, since the server also acts as a test server, the running version may be newer than the one in the repository, as some changes might not have been pushed yet.
:::
