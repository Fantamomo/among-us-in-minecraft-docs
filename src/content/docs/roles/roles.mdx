---
title: Role List
---

This plugin features a variety of roles, each with unique abilities and objectives.

| Role                                    | Description                                                         |
|-----------------------------------------|---------------------------------------------------------------------|
| [Caller](../crewmates/caller)           | Can call meetings from anywhere using a portable meeting button.    |
| [Camera Man](../crewmates/camera-man)   | Can access all cameras from anywhere on the map.                    |
| [Crewmate](../crewmates/crewmate)       | Standard crew member with no special abilities.                     |
| [Detective](../crewmates/detective)     | Can track other players by their footprints.                        |
| [Engineer](../crewmates/engineer)       | Can use vents like an Impostor.                                     |
| [Mayor](../crewmates/mayor)             | Has two votes in meetings instead of one.                           |
| [Seer](../crewmates/seer)               | Can reveal the team of a selected player.                           |
| [Sheriff](../crewmates/sheriff)         | Can kill a player, but risks dying if targeting a Crewmate.         |
| [The Damned](../crewmates/the-damned)   | No abilities, but killers automatically self-report.                |
| [Snitch](../crewmates/snitch)           | Reveals Impostors when all tasks are completed                      |
| [Camouflager](../imposters/camouflager) | Can disguise all players as a single random player.                 |
| [Imposter](../imposters/imposter)       | Can kill players and sabotage ship systems.                         |
| [Miner](../imposters/miner)             | Imposter who can create personal vent networks.                     |
| [Morphling](../imposters/morphling)     | Can morph into other players, copying their name, skin, and helmet. |
| [Phantom](../imposters/phantom)         | Can enter ghost mode to move freely and kill while invisible.       |
| [Arsonist](../neutral/arsonist)         | Wins by dousing all players                                         |
| [Cannibal](../neutral/cannibal)         | Wins by consuming a set number of corpses.                          |
| [Jester](../neutral/jester)             | Wins by being ejected during a meeting.                             |
