The Sheriff can shoot another player.

This ability has the same cooldown as the Impostor’s kill.

The Sheriff can target any player.
- If the target is an Impostor or a Neutral role, the target dies.
- If the target is a Crewmate, both the target **and** the Sheriff die.

The Sheriff must use this ability carefully, as a wrong decision can cost the Crew two lives instead of one.
