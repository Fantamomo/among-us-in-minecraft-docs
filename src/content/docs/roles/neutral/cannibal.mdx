title: Cannibal

import {Badge} from '@astrojs/starlight/components';

<Badge text="Neutral" variant="tip"/>

The Cannibal wins by consuming a specific number of corpses.

When near a corpse, the Cannibal can choose to eat it instead of reporting it.

Once eaten, the corpse is removed from the game and can no longer be reported by any player.

:::tip
The Cannibal has a locator bar marker pointing toward the nearest corpse.
Note that **no notification appears** – the Cannibal must actively watch the bar to detect nearby corpses.
:::
