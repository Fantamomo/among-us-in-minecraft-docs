import { Badge } from '@astrojs/starlight/components';

Type: <Badge text="Short" variant="note" />

In this task, the player must adjust the temperature to match a target value.

**How it works:**
1. Two buttons are provided: **up (red)** to increase the temperature and **down (blue)** to decrease it.
2. The current temperature is displayed in the center along with the **target temperature**.
3. Click the up or down buttons until the current temperature matches the target.
4. Once the temperatures match, the task is completed.

:::tip
The display updates in real-time with each button press, showing your progress toward the target temperature.
:::
