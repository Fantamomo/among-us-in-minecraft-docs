In this task, the player must correctly place artifacts in their designated slots.

**How it works:**
1. The player is given **four artifacts** in the inventory.
2. Drag each artifact to the **matching target slot** below.
3. Once all artifacts are correctly placed, the task is completed.

:::caution
Dragging the wrong artifact or placing it in the wrong slot will mark it red. The task is only completed when all four are correctly positioned.
:::
