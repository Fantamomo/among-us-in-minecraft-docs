In this task, the player must inspect a sample by processing it through several steps.

**How it works:**
1. Press the **start button** to begin the sample processing.
2. A hopper fills the bottles with water.
3. Once all bottles are filled, a timer begins counting down.
4. When the timer reaches zero, one bottle changes color.
5. Click the bottle that changed color to complete the task.

:::caution
Closing the task menu before the timer starts will reset the task. The task also resets if the wrong bottle is clicked.
:::
