import { Badge } from '@astrojs/starlight/components';

Type: <Badge text="Long" variant="note" />

In this task, the player must reboot the Wi-Fi system by activating the reboot sequence.

**How it works:**
1. Click the **reboot button** in the center of the interface to start the reboot process.
2. A timer will start, and a progress bar will fill up around the interface.
3. Wait for the timer to finish. Once complete, the button will turn green.
4. Click the green button to finish the task.

:::caution
Closing the menu during the countdown will pause the visual progress, but the timer will continue running internally. Do not click other slots; only the reboot button affects progress.
:::
