In this task, the player must verify their ID by entering a 7-digit code.

**How it works:**
1. Open the **ID verification panel** to see a 7-slot code row and a 3×3 number grid.
2. Click the numbers in the grid in the correct order to match the code displayed above.
3. Correct selections will briefly highlighted in **green**.
4. Selecting the wrong number will trigger an **error blink** and reset the input.
5. Once all seven numbers are entered correctly, the task is completed.

:::caution
Closing the inventory or clicking the wrong numbers will reset the code.
:::
