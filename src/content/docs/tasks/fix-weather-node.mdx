This is a two-stage task where the player must repair one weather control node.

**Stage 1:**
Flip all the levers in the interface to the “on” position. A visual interface shows the levers and their current state. Once all levers are on, the first stage is complete.

**Stage 2:**
The player moves to a second location and interacts with one of the stage 2 levers to finalize the repair.
Completing this stage finishes the task.
